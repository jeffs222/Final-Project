# Final-Project
Overview

This is a small Tkinter application that lets users enter a stock ticker symbol, view real-time price information, see a history graph, and receive a short AI-generated explanation of the trend. It uses Yahoo Finance for market data, Matplotlib for charting, and Google Gemini for the commentary.

Features

Search any stock ticker

Current price, previous close, and percent change

Price history graphs (1D, 5D, 1M, 1Y)

AI summary of stock movement
